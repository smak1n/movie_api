Documentation Page

To build the server-side component of a “movies” web application. The web
application will provide users with access to information about different
movies, directors, and genres. Users will be able to sign up, update their
personal information, and create a list of their favorite movies.

URL Endpoints

REQUEST URL HTTP METHOD URL PARAMETERS REQUEST BODY DATA FORMAT RESPONSE BODY DATA FORMAT
Return a list of all movies to the user /movies GET /movies None A JSON object holding data about all the movies
Return data about a single movie by title to the user /movies/[Title] GET /movies/:title None A JSON object holding data about a single movie (title, year, genre, director):
{
title: 'Interstellar',
year: 2014,
genre: ['Adventure', 'Drama', 'Sci-Fi'],
director: 'Christopher Nolan'
}
Return data about a genre to the user movies/genres/[Name] GET /movies/genres/:name None A JSON Object holding data about a certain genre(e.g. “Thriller”).
Return data about a director to the user /movies/directors/[Name] GET /movies/directors/:name None A JSON Object holding data about a certain director.
Allow new users to register /users POST /users A JSON object holding data about the user to add, structured like: {
"Username" : "hello_world",
"Password" : "helloworld",
"Email" : "helloworld@world.com"
}		 A JSON object holding data about the user that was added
Allow users to update their user info (username) /users/[Username] PUT /users/:username None A text message indicating the username was updated.
Allow users to add a movie to their list of favorites /users/[Username]/favorites POST /users/:username/favorites None A text message indicating the movie has been added to the user's favorites.
Allow users to remove a movie from their list of favorites /users/[Username]/favorites DELETE /users/:username/favorites None A text message indicating the movie has been removed from the user's favorites.
Allow existing users to deregister /users/[Username] DELETE /users/:username None A text message indicating that a user has been deregistered.
