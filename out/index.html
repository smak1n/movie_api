passport.js

This file implements passport strategies(Passport to implement basic HTTP authentication to log registered users into your application, as well as JWT authentication for subsequent requests to your API.). The "LocalStrategy" takes username and password from the request body and checks its presence in the database collection. For subsequent requests the JWT strategy is used.
